## Getting Started:

### Install Yeoman Package Globally

```
npm i -g yo
```

### Install Generator

```
npm i -g generator-tde
```

<a name="Topic02"></a>

## Generators:

### React-Native Component

```
yo tde:native
```

### React-Native Typescript Component

```
yo tde:tsnative
```

### React Web Component

```
yo tde:web
```

### React Typescript Web Component

```
yo tde:tsweb
```
